# redox-ssh

A ssh client and server written entirely on rust, primarily targeted at (http://redox-os.org).

## Features

Currently implemented features, ordered by priority:

  [ ] SSH Server
  [ ] SSH Client
  [ ] Encryption
  [ ] Public key authentication
  [ ] Port forwarding
  [ ] SCP File Transfers
